# AI Travel Planner

## Overview

AI Travel Planner is a Flask-based web application that helps users plan trips easily.

The application provides:

* Live weather information
* Tourist attractions
* Nearby hotels
* Nearby restaurants
* Transportation options
* AI-generated travel plans
* Google Maps integration

---

## Features

* Weather details using OpenWeather API
* Tourist place recommendations using Gemini AI
* Nearby hotels using Geoapify API
* Nearby restaurants using Geoapify API
* Google Maps navigation links
* Responsive user interface
* AI travel itinerary generation

---

## Installation

Clone repository

git clone  https://saipavankurapati15-ship-it.github.io/ai-travel-planner/

Move to project folder

cd ai-travel-planner

Install dependencies

pip install -r requirements.txt

Run application

python app.py

Open browser

http://127.0.0.1:5000

### OpenWeather API

* Geocoding

* Restaurants
